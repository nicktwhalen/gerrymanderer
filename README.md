# Gerrymanderer

A game that teaches and plays with the concept of gerrymandering.

## Getting Started

### 1. Clone and Install

```bash
git clone <repository-url>
cd gerrymanderer
yarn install
```

### 2. Start frontend server

```bash
yarn dev
```

### 3. Load the game

Navigate to the frontend base URL: http://localhost:3000

## Testing

```bash
# Run unit tests
yarn test
```
